# ToolArc Project Overview

## Overview

ToolArc is a beginner-friendly Japanese blog and tool platform focused on AI workflows, developer tools, and learning resources.

The site aims to help beginners learn modern development workflows using tools like Cursor, Claude, Next.js, GitHub, and Vercel.

---

## Target Audience

- Beginner developers
- Japanese users
- AI beginners
- Users learning Cursor and Claude
- Users building websites for the first time

---

## Main Content

- Cursor tutorials
- Claude usage guides
- AI workflow articles
- Next.js learning articles
- GitHub / Vercel setup guides
- Developer tools
- Gaming-related utilities

---

## Design Philosophy

- Readability first
- Beginner-friendly UI
- White background
- Soft blue accents
- Clean and minimal layout
- Avoid dark cyberpunk SaaS style

---

## Technical Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel
- GitHub

---

## Content Writing Style

- Use simple Japanese
- Explain technical terms carefully
- Focus on step-by-step instructions
- Avoid overly advanced explanations
- Prioritize clarity over cleverness
