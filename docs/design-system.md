# ToolArc Design System

## Theme

- Bright theme
- White base background
- Soft blue accents
- Beginner-friendly
- Clean and minimal

## Colors

- Background: white
- Section background: #f8fbff
- Accent blue: #60a5fa
- Border: #dbeafe

## Style

- Avoid cyberpunk/dark SaaS style
- Focus on readability
- Blog-first design
- Large spacing
- Rounded corners

## Target

Beginner developers and AI users
