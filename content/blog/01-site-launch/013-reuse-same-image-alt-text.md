# 同一画像を記事内で再利用するときはaltテキストを変える

記事の途中で「さっき出した比較図、もう一度見せたいな」という場面、よくありますよね。手順の振り返りや、別セクションでの再確認のために同じ画像を使い回すこと自体は珍しくありません。ただ、その際に「同じ画像をそのまま貼って大丈夫なのか」「SEOに悪い影響はないのか」と気になった方もいるのではないでしょうか。

筆者も同じ画像を再掲する場面で同じ疑問を持ちましたが、画像そのものの再利用より、**altテキストとcaptionの完全コピー**のほうが気をつけるべき点だと考えられます。

> **今日の結論**
> - 同一ページ内や記事間で同じ画像を再利用しても、SEOペナルティになることは一般的には少ないです
> - ただし alt テキストと caption を完全コピーすると、内容が薄い印象になりやすいです
> - 2回目以降の掲載では alt を文脈に合わせて変え、直前に振り返り1行を足すと読みやすくなります
> - サイト全体で同一 alt の量産貼り付けは避けます

## 同じ画像を再利用してもSEOペナルティになりにくい理由

画像ファイル自体の重複利用は、記事の構成上よくあることです。同じ比較図や手順スクリーンショットを、説明の流れに合わせて複数回見せること自体に大きな問題はないと考えられます。重要なのは画像ファイルそのものではなく、**その画像をどう説明しているか**という部分です。

## alt・captionを完全コピーすると起きる問題

ここで注意したいのが、1回目の掲載と同じalt・captionをそのまま2回目にも使ってしまうケースです。文章としても「同じ説明を繰り返しているだけ」に見えやすく、読者にとっても内容が薄い印象になりがちです。検索エンジンから見ても、同一の説明テキストが複数箇所にあることは、ページの情報密度が低く見える一因になり得ると考えられます。

## 2回目以降の掲載で意識する書き方

筆者が公開しているスマホ移行手順の記事（[ChatGPTのスマホ移行手順](/blog/chatgpt-smartphone-account-migration-tips)）でも、同じ比較図を別セクションで再掲する場面がありました。その際に意識したのは次の点です。

1. 1回目の掲載では、そのセクションの説明に合った alt を書く
2. 同じ画像を別セクションで再掲する場合は alt を変える（例: 移行前後の比較画面 → 上記比較図のおさらい）
3. 2回目の直前に「先ほどの比較図のおさらいです」など1行の振り返り説明を足す
4. 1回目とまったく同じ文言にならないよう、視点や目的を少し変える

こうすることで、同じ画像でも「なぜ今もう一度見せているのか」が読者に伝わりやすくなります。

## サイト全体で同一altを量産しないための運用ルール

1記事内だけでなく、サイト全体で同じaltテキストを量産的に貼り付けることも避けたいポイントです。テンプレート的に同じ説明文を使い回すと、画像とテキストの関係が形だけになりやすくなります。新しい記事で画像を使う際は、「この画像にはこの記事・このセクション専用の説明になっているか」を一度確認するとよいと考えられます。

## チェックリスト

| 確認項目 | 状態の目安 |
| --- | --- |
| 同じ画像を別セクション・別記事で使っているか | 使っている場合は次の項目を確認 |
| altテキストが1回目と完全に同じか | 同じなら文脈に合わせて書き分ける |
| captionが1回目と完全に同じか | 同じなら振り返り用の文言に調整する |
| 再掲の直前に1行の振り返り説明があるか | ない場合は読者が「なぜ再掲か」分かりにくい |
| サイト全体で同じalt文言を量産していないか | 量産している場合はテンプレート化を見直す |

## まとめ

同じ画像の再利用そのものは問題になりにくいものの、altテキストとcaptionの完全コピーは内容が薄く見える原因になりやすいです。再掲時はaltを文脈に合わせて変え、直前に1行の振り返りを添えることで、読者にとっても自然な構成になります。
