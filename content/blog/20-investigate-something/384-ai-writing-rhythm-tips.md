# AIっぽい文章は語彙よりリズムで見抜かれる｜文末3連続と段落の均質さを直す

「AIっぽい文章」と言われると、多くの人がまず語彙を疑います。「まさに」「以下のように」「〜という点も見逃せません」といった、いかにも生成AIが好んで使う言い回しを削れば直る、という発想です。

ところが、それらの言葉を一通り削っても、なぜか読み手には違和感が残ることがあります。原因は語彙ではなく、文のリズムにあることが多いためです。具体的には、**文末表現の均質さ**と**段落構造の均質さ**の2点。この記事では、この2点をどう直すかを短く整理します。

> **今日の結論**
> - AIっぽさの正体は語彙よりリズム。同じ文末（です/ます等）の3連続と、段落構造の均質さが主な原因
> - NGワードを機械的に削っても、リズムが均質なままなら違和感は残りやすい
> - 直し方は「文末の多様化」「段落の密度に波をつける」「体言止めは山場だけに使う」の3点
> - 「体言止め◯%」のような数値目標で管理しようとすると、かえって不自然なパターンになりやすい

## なぜNGワード狩りだけでは直らないのか

AI検出を意識した文章添削は、特定の単語やフレーズを機械的に置き換える形で進められがちです。単語レベルの修正は目に見えて分かりやすく、着手もしやすいという事情があります。

ただ、読み手が「AIっぽい」と感じる判断は、単語を1つずつ意識して読んだ結果ではありません。文末が同じ調子で続く、段落の長さと型がずっと揃っている。そうした構造的な均質さを、無意識のうちに拾っているのが実情です。単語を差し替えても、この構造がそのままなら印象はあまり変わりません。

## 文末の3連続をチェックする

まず見るべきは、「です」「ます」のような同じ文末表現が3文以上続いていないかどうかです。

例えば次のような文章。

> この機能は便利です。設定も簡単です。誰でも使えます。

文法的には何の問題もありません。ただ、3文とも同じ「〜す」で終わっており、読んでいて単調に感じます。文末を崩すと、印象はこう変わります。

> この機能は便利です。設定も簡単。迷わず使えるはずです。

2文目だけ体言止めにして、拍を変えています。全部の文で体言止めを使う必要はなく、区切りや山場になる1文だけで十分です。

## 段落の均質さを崩す

文単位だけでなく、段落単位でも同じことが起きます。「状況→理由→対策」という型を、毎段落まったく同じ順番・同じ長さで繰り返すと、内容が変わっても構造が単調に見えてしまいます。

対策は、密度の高い段落が2〜3個続いたら、あえて短い段落を1つ挟むことです。この段落自体がその例に当たります。読み手にとっての「一息つく場所」になり、次の段落をまた読み進めやすくなります。

## 数値目標を設けると逆に不自然になりやすい

「体言止めを60%にする」「常体と敬体を◯：◯の比率にする」といった数値目標を見かけることがあります。管理はしやすそうですが、実際にやってみると、目標達成のために不要な箇所まで体言止めにする、という逆効果が起きやすくなります。

筆者は2026年7月、公開済み記事を読み返しながら文末パターンを見直す作業をしました。その中で気づいたのは、同じ文末が3文以上連続している箇所がいくつかの記事に見られた、ということです。具体的な該当記事数や比率までは確認できていないため、ここでは数値化していません。数値よりも、「3連続していないか」「山場で体言止めを使えているか」という具体的なチェック観点のほうが、実際の直しには使いやすいという実感です。

## 確認しておきたいチェックリスト

| 確認項目 | 状態 |
| --- | --- |
| 同じ文末（です/ます等）が3文以上連続していないか | |
| 段落の型（状況→理由→対策）が毎回同じ長さ・同じ順番で続いていないか | |
| 体言止めが山場以外で多用されていないか、逆にゼロになっていないか | |
| NGワードを削っただけで満足せず、文末・段落構造まで見直したか | |
